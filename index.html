Login with SelfKey Chrome Extension alpha

Login with SelfKey allows users a way to securely and privately authenticate themselves without the need for a centralized authority. In order to use the alpha version of the extension, please follow these instructions:

  1. Download the source code for the extension below and extract the files
  2. Goto chrome://extensions in your Chrome browser
  3. Enable "Developer Mode"
  4. Select "Load Unpacked Extension" and choose the "/src" directory of the extracted files
  5. Make sure the extension is enabled - you should see the SelfKey icon in your browser
  6. Navigate to a participating website and follow the instructions to log in
